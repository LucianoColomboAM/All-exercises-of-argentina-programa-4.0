Etiquetas

abbr:
Es para que muestre un texto si pones el mouse en las palabras abreviadas
h1,2,3,3,etc:
Son titulos, a medida que se agranda el numero es mas chica la letra
p:
Es para parrafos
  • Atributo lang:(lang="es") Es para indicar el lenguaje del parrafo
  • Atributo class: (class= "prueba") define una clase de parrafos (p) a la que se puede referir luego
br:
La etiqueta br produce un salto de línea en el texto (retorno de carro).No tiene etiqueta de cierre, lo que está a continuación de la etiqueta , se mostrará en la siguiente línea.
strong:
La etiqueta define texto con fuerte importancia. Usualmente se verá en negrita.
